## Run the app

You have to run the server and the client app to see this example working.

### Server

This is the server with a GraphQL API.

Run it!

1. `npm install`
2. `npm start`

### Client

The client app is all about the user interface.

Run it!

1. `cd client`
2. `ng serve`
